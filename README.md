# crypto-pnl-analyzer
Track crypto P&amp;L across wallets with our Python/Node.js &amp; React analyzer. Using FIFO/LIFO/HIFO, it provides IRS-compliant tax reporting, DeFi/staking support, and wash sale detection. Store data in PostgreSQL and export to PDF/CSV. This tool offers real-time analytics for traders and tax pros managing digital asset portfolios.
